# KaoYanBao
考研宝SSM项目
我这是学校组织的专业实训中完成的项目，我负责项目全部后台的编写，以及前端页面的调试和优化，数据库结构设计，系统规划和设计，项目组里4个人，只有我负责后台，因为大家都比较菜，所以代码质量不高，只是实现了需求中提出的所有功能，可拓展性和可维护性还需要进一步打磨。
下面是项目介绍
### 项目描述：
目前考研的人群越来越多，考研宝为考研人群提供考研信息服务，由于每个高校对考研的具体科目以及要求不一样，而且每年各科目的分数线和具体的教材也会有所不同，学生在考研准备的同时还需要花费大量的时间去进行此类相关信息的收集。考研宝通过爬虫，对某个高校某个院系的信息进行获取并进行整理，便于学生查询。同时对于具体的学习资料进行整理，便于学习。  
### 项目功能：
主要功能模块包括：
1. 用户模块：
用户模块包括用户都登录和注册，通过输入手机号来进行网站的注册登录。
2. 个人中心模块：
个人中心中有个人的相关信息资料修改，以及个人收藏，可以查看或收藏的个人链接。
3. 信息查询模块： 
3.1 查询信息：	搜索考研相关信息及学习资料
3.2 添加收藏：	将搜索得到的信息链接添加到“个人收藏”
3.3 浏览推送	：	查看网站首页推荐的信息资料
4. 后台管理：
4.1 用户管理，对注册用户进行封禁管理，以及搜索查看具体用户；
4.2 网站信息管理，对于网站首页推荐的信息资料进行管理，能够进行增删改查相应操作；
4.3 数据统计，能够针对网站用户收藏浏览信息进行统计，分析用户行为。
5. 爬虫模块实现
可以在后台中，通过提供的模块对高校页面等进行爬取
